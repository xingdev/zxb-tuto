##命令行程序demo
- 加入执行权限
```
chmod +x ~/tuto/node/l1/bin/node-echo.js
```

- 建立软链接

```
sudo ln -s ~/tuto/node/l1/bin/node-echo.js /usr/local/bin/node-echo
```

- 执行 node-echo
